Embeds Vapi Voice AI Assistant into WordPress with customizable button settings and appearance options.

== Description ==

Vapi Voice AI Agent is a WordPress plugin that seamlessly integrates Vapi's voice AI technology into your website. With this plugin, you can add an interactive voice assistant that visitors can use to get information, ask questions, or interact with your content using natural voice commands.

= Features =

* Easy API integration with Vapi platform
* Customizable voice button appearance and position
* Multiple button states (idle, loading, active) with different styling
* Responsive design that works on all devices
* Admin dashboard with configuration options
* Training section for customizing AI responses
* Tools for plugin management and diagnostics

= Requirements =

* A Vapi account and API key
* Assistant ID from your Vapi dashboard
* WordPress 5.0 or higher
* PHP 7.4 or higher

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/vapi-voice-ai-agent` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to 'Vapi Agent' in your WordPress admin menu.
4. Go to 'Vapi Configuration' > 'API Config' tab.
5. Enter your Vapi API Key and Assistant ID.
6. Customize the appearance in the 'Appearance' tab if desired.
7. Save your settings and visit your website to see the voice assistant button.

== Frequently Asked Questions ==

= Where do I get my Vapi API Key and Assistant ID? =

You need to create an account at [Vapi.ai](https://vapi.ai) and create an assistant. Your API key and Assistant ID will be available in your Vapi dashboard.

= Can I customize the button appearance? =

Yes! The plugin provides extensive customization options including button position, colors, icons, and text for different states (idle, loading, active).

= Does this work on mobile devices? =

Yes, the voice assistant works on both desktop and mobile devices, provided the device supports voice input.

= Is there a way to train the AI assistant? =

The plugin includes a Training tab where you can add training notes and configure your assistant's responses. The actual training is done through your Vapi dashboard.
